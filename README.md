# Reliance Tools API
This is a collection of tools for flash updating, configuring, and maintaining your Reliance Thermal Printer.

# Installation
### Package Manager  

    Install-Package PTIRelianceLib
    
### Dotnet CLI

    dotnet add package PTIRelianceLib
